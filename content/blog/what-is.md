我最近一直在思考“什么是什么”这样的问题，因为我想要对核心概念有自己的定义、自己的标准   
这样的问题，最好的一个聊天对象就是 gpt ，通过和它聊了很多轮之后，写了一个提示词

```
是什么？
对一个概念做一句话的本质性描述

一条完整的定义包含两部分：
定位：它是什么
边界：它不是什么

---

核心步骤
步骤 1：先写出概念的一句话本质定义
这句话应当回答：
“它究竟在做什么？”
“它让世界发生了什么改变？”
不需要追问细节，只要一句话抓住本质动作
例如：
消费 = 用当前的资源换取当下的体验或满足
投资 = 用当前的资源换取未来产生更多资源的可能性

---

步骤 2：从定义句中挑出关键词
将定义句拆成几个核心词：
例如：
消费：用 / 当前的资源 / 换取 / 当下 / 体验或满足
投资：用 / 当前的资源 / 换取 / 未来 / 产生 /更多资源 / 可能性

---

步骤 3：对每个关键词逐一追问「是什么？」
关键点：
- 每次只追问当前定义句中的词，不深入词的词
- 不要无限往下挖
- 停在你能清晰表达的最原子动作
例如：
 “资源是什么？”
 → 可以被交换、使用、消耗来改变状态的东西
“未来是什么？”
 → 还没发生、但会受到现在行为影响的时段

---

步骤 4：将所有追问整合为“展开定义”
这是一种类似“分形”的结构：
投资 =
- 把资源（=可被交换或使用的东西）
- 从现在（=正在发生）
- 移向未来（=还没发生但会受影响）
- 并且期待回来更多资源（=净正回报）

---

步骤 5：判断定义是否完成
判断方式只有一个：
如果换成 5 岁小孩都能明白的句子，你仍然认可它，这个定义就完成了

---

定义术的黄金法则
法则 1：不从词典取定义，从体验取定义
不接受“书本答案”
定义必须来自：
- 自己的观察
- 自己的经验
- 看到世界的方式

---

法则 2：定义必须聚焦于“动作”
坏定义：描述它是什么东西
好定义：描述它在做什么、带来什么变化
例：
 ❌ 投资是为了赚钱
 ✔ 投资 = 把资源从现在移向未来，让未来的资源变更多

---

法则 3：定义尽量越短越好
越短：
- 越锋利
- 越能指导行动
- 越能变成你自己的语言系统

---

法则 4：定义必须能判断“算 / 不算”
能够判断边界：它是不是投资？它是不是浪费？它是不是消费？

---

法则 5：定义不是永久的，而是“当前最有用版本”
定义是一种工具，不是教条
可以随时更新自己的定义

```

这是我打磨的一套提示词，把这套提示词丢给 gpt，然后再问它，“什么是什么”这样的问题，就可以得到一个 70 分的答案，再结合自己的经验、经历什么的，就可以把“什么是什么”这样的问题，打磨到 80 分左右

这是我基于上面这套提示词，问的几个概念，它给我的定义（我忘记后来我有没有继续打磨了）

### 消费 是什么？   
消费 = 用当前的资源换取当下的体验或满足

用 = 把资源从当前状态转移出去   
当前的资源 = 此刻能够调动的金钱、时间、注意力、体力、情绪、技能等   
换取 = 把可用资源置换成另一种   
当下 = 在现在这个时间窗口里立即发生   
体验或满足 = 让当前的感受、情绪、需求被回应   

### 投资 是什么？   
投资 = 用当前的资源换取未来产生更多资源的可能性

用 = 把资源从当前状态转移出去   
当前的资源 = 此刻能够调动的金钱、时间、注意力、体力、情绪、技能等   
换取 = 把可用资源置换成另一种   
未来 = 不在当下，稍后才会发生的一段时间   
产生 = 让资源出现、增加或成长   
更多资源 = 回到手上的金钱、技能、关系、机会等的增量   
可能性 = 不保证，但具有概率性或预期性的结果

### 浪费 是什么？   
浪费 = 用当前的资源换取既不带来当下满足、也无法带来未来回报的结果

用 = 把资源从当前状态转移出去   
当前的资源 = 此刻能够调动的金钱、时间、注意力、体力、情绪、技能等   
换取 = 把可用资源置换成另一种   
结果 = 某个发生的状态、产物或体验   
不带来当下满足 = 你当前的需要没有被回应   
也无法带来未来回报 = 不会让以后资源变多，也不会创造价值或可能性

### 财富 是什么？   
财富 = 让你未来的选择变多、行动更自由的那些资源

未来 = 还没发生、但会被现在的行为改变的时间段   
选择变多 = 在同样的情境下，拥有更多可能行动的路径   
行动更自由 = 可以做想做的事，而不被强迫做不想做的事   
资源 = 能被使用、交换、消耗，以改变自己或外部世界状态的东西（钱、技能、时间、关系、健康……都算）

钱是财富（因为增加选择）   
技能是财富（因为减少被迫）   
好身体是财富（因为行动自由）   
自动复利系统是财富（选择更多）   
- 显摆不是财富   
- 昙花一现的快感不是财富   
- 束缚你的东西不是财富

### 金钱 是什么？   
金钱 = 让人们快速、广泛且精确地交换价值的工具

快速 = 减少达成交换所需要的时间   
广泛 = 让交换发生在更多人、更多场景之间   
精确 = 更好地衡量与匹配交换双方的价值   
交换 = 双方用各自拥有的东西，换取对方拥有的东西   
价值 = 对某人某刻有帮助或让状态变更好的东西   
工具 = 提升效率

### 自我 是什么？   
自我 = 你对当下自己的持续认知与选择能力

你 = 外界与内在经验的体验者   
当下的自己 = 这一刻你的想法、情绪、身体感受、冲动、欲望、判断、行为的整体，是持续流动的状态   
认知 = 对当前状态的观察、理解与解释的动作   
选择 = 在多种可能的行动路径里挑一个   
能力 = 能反复执行某种动作的稳定性
